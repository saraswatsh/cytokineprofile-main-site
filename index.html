Citing CytokineProfileShinyApp

The goal of CytokineProfile Shiny is to conduct quality control using biologically meaningful cutoffs on raw measured values of cytokines. Specifically, it tests distributional symmetry to suggest transformations, performs exploratory analysis (summary statistics, barplots, boxplots), and runs univariate and multivariate analyses for advanced insights. It provides a user-friendly wizard-style interface for your own data. The underlying R package is available on GitHub and CRAN.

The Shiny application is available at the CytokineProfile Shiny App.

Citation

Citing CytoProfile

CytoProfile is an R package designed for advanced cytokine data analysis. It provides a comprehensive suite of functions for exploratory, univariate, and multivariate analysis, as well as machine learning tailored to your cytokine datasets.

Key Features

  • Exploratory Analysis: Bar plots, boxplots, summary statistics.
  • Statistical Testing: t-tests, ANOVA, univariate methods.
  • Multivariate Analysis: PCA, PLS-DA, etc.
  • Machine Learning: Random Forest, XGBoost.
  • Data Handling: Seamless loading and preprocessing.

Install the development version from GitHub:

# install.packages("devtools")
devtools::install_github("YOUR_USERNAME/CytoProfile")

Install from CRAN:

install.packages("CytoProfile")
